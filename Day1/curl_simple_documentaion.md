## Curl Documentation

### Basic Usage
```bash
curl [options] URL
```

### Common Options

#### -i (--include)
- Shows the HTTP response headers 
- Example:
```bash
curl -i https://api.github.com/users/MohamedElshishtawy
```

#### -v (--verbose)
- Provides detailed information
- Example:
```bash
curl -v https://api.github.com/users/MohamedElshishtawy
```
